03 ArrayList vs LinkedList 使用场景

📖核心概念

ArrayList 和 LinkedList 都是 Java 集合框架中 List 接口的实现，但底层数据结构和性能特点完全不同。

  • ArrayList：基于动态数组实现，支持随机访问
  • LinkedList：基于双向链表实现，不支持随机访问
🎯 关键区别
ArrayList 适合读多写少，LinkedList 适合频繁插入/删除

⚖️性能对比

操作ArrayListLinkedList
随机访问 get(i)O(1) ✅O(n) ❌
尾部追加 add(e)O(1) 均摊 ✅O(1) ✅
头部插入 add(0,e)O(n) ❌O(1) ✅
内存占用较低较高

🎤面试要点

  • ArrayList 扩容机制：默认 10，1.5 倍扩容，频繁扩容影响性能
  • 遍历性能：ArrayList 比 LinkedList 快（CPU 缓存友好）
  • 线程安全：都不线程安全

💥踩坑经验

🚫 for循环遍历中删除元素
索引错乱。正确做法：① Iterator.remove() ② 从后往前遍历 ③ removeIf()
